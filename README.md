# Playwright OrangeHRM Automation

This project automates the **User Management** module of OrangeHRM using **Playwright** with JavaScript.  
It covers:
- Login automation
- Adding a new user
- Searching users

## Tech Stack
- **Playwright** (JavaScript)
- Node.js
- Git & GitHub
